ABOUT THE PROJECT

Video Game Diary

Video Game Diary is a Letterboxd-style Angular application for discovering, searching and tracking video games.

Discover

Search for games using the RAWG API and view detailed information about each title.

Track

Save games to a personal library and organise them as Played, Playing, or Want to Play.

Rate

Add a personal rating when saving games, making the app feel closer to a real diary system.

Technology Used

Angular TypeScript Bootstrap RAWG API Node.js Express MongoDB
